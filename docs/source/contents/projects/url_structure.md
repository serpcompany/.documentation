# URL Structure - STANDARDIZED

## CORE
- `site.com/`
- `site.com/about/`
- `site.com/contact/`
- `site.com/legal/`
- `site.com/legal/privacy-policy/`
- `site.com/legal/terms-conditions/`
- `site.com/legal/affiliate-disclosure/`
- `site.com/legal/dmca/`
- `site.com/partner/`
- `site.com/category/<keyword>/`

## BLOG
- `site.com/blog/`
- `site.com/blog/<single>/`
- `site.com/blog/c/<category>/`

---
